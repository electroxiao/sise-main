### 2.1 牌库构成（117张）
| 类型 | 内容 | 数量 | 特殊规则 |
|------|------|------|----------|
| **基础牌** | 将、士、象、车、马、炮、卒 × 4色（黄/红/绿/白）× 每组合4张 | 112张 | “将”可组牌但**不可碰、不可主动打出** |
| **金条牌** | 公、侯、伯、子、男（各1张，视作特殊的红色） | 5张 | **定庄时视为红色**；功能同“将”，**不可碰、不可主动打出** 


### 定庄
首局随机一人翻牌，根据牌面颜色决定，黄=1, 红=2, 绿=3, 白=4点，暨如果翻到牌的颜色是黄色则自己是庄。
大胡后，胜利方的对家成为新庄。
小胡后，庄家连庄。

### 💰 结算规则（精准版）
#### （1）胡牌得分

胡牌条件：手牌+明示区都是有效的牌组，无落单牌
**有效牌组清单**：  
| 牌组     | 组成                   | 胡牌计分 |
| -------- | ---------------------- | -------- |
| 车马炮架 | 同色车+马+炮           | 1        |
| 将士象架 | 同色将+士+象           | 1        |
| 三异色卒 | 3色不同“卒”            | 1        |
| 四异色卒 | 4色不同“卒”            | 2        |
| 对子     | 同色同字两张           | 0        |
| 单将组   | 手牌中单独1张“将”      | 1        |
| 单金条组 | 手牌中单独1张“金条”    | 3        |
| 普通坎   | 3同色同字（非将/金条） | 3        |
| 金条坎   | 任意3张金条            | 9        |
| 普通开   | 暗坎+响应第4张         | 6        |
| 金条开   | 3金条+响应第4张        | 18       |
| 普通鱼   | 亮出的4同色同字        | 8        |
| 金条鱼   | 亮出的4/5张金条        | 24       |

| 类型 | 公式 | 说明 |
|------|------|------|
| **小胡** | `3 + 吃分 + 碰分 + 未开坎分 + 单张将/金条分` | 无任何“鱼”或“开” |
| **大胡** | `(3 + 吃分 + 碰分 + 未开坎分 + 开分 + 鱼分 + 单张将/金条分) × 2` | 含至少1个“鱼”或“开”（含金条开/鱼） |

#### （2）互相结算（非胡牌玩家参与）
| 组合类型 | 每组结算分 | 说明 |
|----------|------------|------|
| 普通坎 / 将坎 | 3分 | 亮出或明示区中 |
| 金条坎 | 9分 | |
| 普通鱼 / 将鱼 | 8分 | |
| 金条鱼 | 24分 | |
| 普通开 / 将开 | 6分 | **新增：开操作参与互结** |
| 金条开 | 18分 | **新增：金条开互结分** |
| 碰、吃、单张组 | 0分 | 仅计入胡牌得分，不参与互结 |

> 📌 示例：玩家A有1个金条开（18分）+1个普通坎（3分），则B/C/D中不是最终赢家的每人付A 21分  
> 📌 流局时：仅执行“互相结算”，无胡牌得分
